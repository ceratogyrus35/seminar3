## Списки

Чтобы добавить ненумерованные списки , необходимо выделить пункты звездочкой (*).
Например, вот так: 
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованые списки, необходимо пункты просто пронумеровать.

1. Первый пункт
2. Второй пункт 
3. Третий пункт
